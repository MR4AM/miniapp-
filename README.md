# miniapp-
##基于wepy小程序框架开发的小程序通用组件市场
### 基于wepy 搭建
- npm install wepy-cli -g  全局安装或更新WePY命令行工具
- wepy init standard projectname 创建项目名为projectname的项目
- cd projectname 进入项目目录
- npm i 安装依赖包
- npm run dev 运行编译测试环境
- npm run bulid  运行编译正式环境
### 启动
- npm i
- npm run dev 运行编译测试环境
- npm run bulid  运行编译正式环境
